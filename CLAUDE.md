## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues (`asphaltbuffet/advent-of-code`), managed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

Triage is not used in this repo; issues are not run through the triage state machine. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root (created lazily by `/grill-with-docs`). See `docs/agents/domain.md`.
